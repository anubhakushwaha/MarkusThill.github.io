Interesting Properties of the Exclusive Or (XOR)

The exclusive-OR – sometimes also exclusive disjunction (short: XOR) or antivalence – is a boolean operation which only outputs true if only exactly one of its both inputs is true (so if both inputs differ). There are many applications where the XOR is used, for instance in cryptography, gray codes, parity and CRC checks and certainly many more. Commonly, the symbol is used to denote the XOR operation. Here, we will use the symbol for the Exclusive-OR and for its negation, the biconditional operator. In this blog post we will look at a few of its interesting properties which can be useful.
